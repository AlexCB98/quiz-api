# Quiz API

A Python quiz application that fetches True/False questions from the Open Trivia Database API and displays them through a graphical user interface built with Tkinter.

This project was built using object-oriented programming (OOP), API requests, JSON data, and Tkinter.

---

## Features

* Fetches quiz questions from the Open Trivia Database API
* Displays questions in a Tkinter graphical interface
* Supports True / False answers
* Tracks the player's score
* Provides instant visual feedback (green/red)
* Automatically loads the next question
* Detects when the quiz is finished
* Decodes HTML entities returned by the API
